It is created for learning purposes and follows the Java Backend Junior roadmap.

- Handle orders, balance, and stock
- Custom exceptions for invalid operations
- Organized package structure

## Technologies

- Java Core
- Object-Oriented Programming (OOP)
- Exception Handling
